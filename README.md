# vue-template-pc
vue脚手架-模板

### 启动方式:

* 将项目拉到本地,安装依赖(npm i); 建议装上yarn 和 nrm
* 开发环境启动项目  npm run dev
* 项目打包 npm run build
* 项目指令化创建页面 npm run create-page  详细脚本代码 /template/script.js  和/template/util.js
